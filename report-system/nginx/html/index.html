Simple Report API

1. Ogólny opis aplikacji

Aplikacja to system służący do zgłaszania awarii w budynkach, umożliwiający mieszkańcom szybkie przesyłanie zgłoszeń dotyczących usterek, problemów technicznych oraz uszkodzeń infrastruktury.

System pozwala administratorom na zarządzanie zgłoszeniami, monitorowanie statusu napraw oraz przypisywanie awarii do odpowiednich pracowników lub serwisów technicznych.

2. Użyte technologie
  • Java 21
  • Maven
  • Spring Boot
  • SpringSecurity (JWT)
  • Docker
  • Postrgres
  • Nginx
